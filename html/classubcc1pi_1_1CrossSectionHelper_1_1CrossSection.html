std::shared_ptr< TH1F > ubcc1pi::CrossSectionHelper::CrossSection::GetReweightedFlux ( const std::shared_ptr< TH1F > &  nuEnergyUniverse) const
private

Get a reweighted flux distribution according to the input neutrino energy spectrum (for the desired universe)

In each true neutrino energy bin, this function finds the ratio of the total number of neutrino events in the supplied universe to the nominal universe. The number of events in a given energy bin depends on the cross-section and the flux at that energy. If the input universe differs only in the flux to the nominal universe, then this ratio gives the fractional change in the flux. This is then used to reweight the nominal flux distribution.

Parameters
nuEnergyUniversethe total neutrino energy spectrum in the desired universe
Returns
the reweighted flux distribution
